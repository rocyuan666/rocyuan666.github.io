【vue】vue3-虚拟列表组件

思路

只渲染可视区域的数据项

  • 滚动时动态计算开始项的索引。
  • 通过开始项索引计算出结束索引。
  • 有了开始结束索引，使用 slice 切割原始数组，得到新数组（实际渲染的数据）。

滚动条的实现

  • 使用一个空白的占位元素，通过计算数据总高度，设置其高。

内容始终在中间

  • 滚动时记录偏移量，使用 position: top 定位 或者 transform: translate 让可视区域始终在中间，不会滚动上移。
